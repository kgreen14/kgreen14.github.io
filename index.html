CROWDed

This app, titled 'Crowded' is designed to allow a user to view a list of places; Ponce City Market, Clermont Lounge, Victory, etc... The user can click on a particular place to view ratings and comments for the place; to make an informed decision on whether they want to patronize the place. The goal is to have comments indicate through a range of 1-5 how crowded a place is. They can also create a profile and comment themselves.

  • Deployed App: Check out CROWDed
  • Node/Express + Mongo/Mongoose: GitHub
  • Build Time: Group 5 Days

Watch-Next

For my final project I decided to create an app that solves a real problem my wife and I have. We love watching movies, but we've seen so many that its hard decide what to watch next. The goal for this app is to be able to search for movies and add them to your favorites list, then have movies recommended to you. Using the OMDBapi I was able to request movie data and store/show the relevent information in a user's favorites list.

  • Deployed App: Check out Watch-Next
  • React + Ruby on Rails: GitHub
  • Build Time: Solo 5 Days
